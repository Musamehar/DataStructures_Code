# DataStructures-Code

## Types of Code

### Data Structures
- **Singly Linked List** (`singly.cpp`) - Linear data structure with forward traversal
- **Doubly Linked List** (`doubly.cpp`) - Bidirectional linked list with forward and backward pointers
- **Circular Linked List** (`circular.cpp`) - Linked list where the last node points back to the first
- **Stacks** - Three different implementations:
  - Static array-based stack (`stack with static array.cpp`)
  - Dynamic array-based stack (`stack with dynamic array.cpp`)
  - Linked list-based stack (`stack with linked list (singly).cpp`)

### Algorithms & Array Operations
- **Array Operations** (`arrays.cpp`) containing:
  - Static and dynamic 2D arrays
  - Matrix memory mapping (row-major, column-major)
  - Recursion examples (factorial, Fibonacci)
  - Tower of Hanoi puzzle
  - Triangular and diagonal matrix operations
